Usage

Send a POST requeST to the following endpoint:
https://collections.clarin.eu/submit/{api version}/{type}

Where:

  • {api version} is the number of any of the currently supported versions:
    • 1.0
    • 1.1 (latest)
  • {type} is:
    • extensional, to submit an extensional collection.
    • intensional, to submit an intensional1 collection.
    More information about the collection types can be found on the help page.

Supported parameters:

Parameters must be form url encoded (application/x-www-form-urlencoded), more info on MDN Webdocs.
Name Type Required Endpoint support
name String Required Extensional + Intensional
description String Required Extensional + Intensional
keyword List(String) Required Extensional + Intensional
purpose Controlled Vocabulary Optional Extensional + Intensional
reproducibility Controlled Vocabulary Optional Extensional + Intensional
reproducibilityNotice String Optional Extensional + Intensional
metadataUri List(String) or List(JSON) Required Extensional
resourceUri List(String) or List(JSON) Required Extensional
queryDescription String Required Intensional
queryUri String Required Intensional
queryProfile String Required Intensional
queryValue String Required Intensional

Example

Basic example: 
curl -v \
    -d 'name=test&
        metadataUri=http://www.clarin.eu/1&
        metadataUri=http://www.clarin.eu/2&
        resourceUri=http://www.clarin.eu/&&
        description=test-collection&
        keyword=test&
        purpose=&
        reproducibility=none' \
    http://localhost:8080/vcr/service/submit/extensional
Supply metadata for references: 
curl -v \
    -d 'name=test&
        metadataUri={"uri": "http://www.clarin.eu/1", "label": "reference 1", "description": "Testing the first reference"}&
        metadataUri={"uri": "http://www.clarin.eu/1", "label": "reference 2", "description": "Testing a second reference"}&
        description=test-collection&
        keyword=test&
        purpose=&
        reproducibility=none' \
    http://localhost:8080/vcr/service/submit/extensional
