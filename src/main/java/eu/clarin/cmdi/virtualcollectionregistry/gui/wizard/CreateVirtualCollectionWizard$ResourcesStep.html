Add resource or metadata references

Click add to add an individual reference of the type 'Metadata' or 'Resource' and optionally add a label and/or description.

Click add more to add multiple references (one per line) of one type without labels or resources. Individual resources can be edited at any stage.

It is recommended to only use persistent identifiers as references. The following are examples of valid persistent identifiers:

  • hdl:1839/00-0000-0000-0001-305B-C
  • doi:10.10.1038/nphys1170
  • http://hdl.handle.net/1839/00-0000-0000-0000-0000-4
  • http://dx.doi.org/10.1002/0470841559.ch1
