About the Virtual Collection Registry

The Virtual Collection Registry (VCR) was developed for CLARIN at The Institute for the German Language (IDS) and the CLARIN ERIC .

The VCR was developed by:

  • Oliver Schonefeld (IDS)
  • Twan Goosen (CLARIN ERIC)
  • Willem ELbers (CLARIN ERIC)

Persistent identifiers for the VCR are issued by GWDG using the EPIC API version 2.
